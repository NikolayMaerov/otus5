Описание свойств организации.

Заполните сведения об организации:

  • Наименование - название организации, которое будет использоваться в программе при выборе организации.
  • Префикс - уникальный префикс, который будет автоматически добавляться программой в начало номера (или кода) при вводе новых документов (элементов списков). Элементы списков, документы и другие объекты программы нумеруются последовательно. Для того чтобы документы по различным организациям отличались по номерам, а элементы списков - по кодам, используется префикс.

Если в программе используется список Виды контактной информации, то для организации будут доступны предопределенные реквизиты для хранения контактной информации:

  • Поля Почтовый, Юридический, Фактический адрес можно заполнить вручную или по шаблону (только для российских адресов, при этом для ввода по шаблону необходимо загрузить адресный классификатор). Для заполнения по шаблону нажмите Выбрать, заполните адрес, нажмите ОК. Если адреса повторяются, то можно выбрать из выпадающего списка с помощью кнопки   Выбрать из списка.  
  • Поля Телефон и Факс можно ввести вручную или по шаблону. 
  • В поле E-mail введите адрес электронной почты организации. Программа проверяет правильность ввода. С помощью кнопки  можно отправить по введенному адресу исходящее электронное письмо.
  • В текстовое поле Другое длиной 1024 символа можно ввести какую-либо дополнительную информацию об организации, например, как добраться до офиса, в котором она находится.

Если при настройке одного или нескольких Видов контактной информации включен флажок Разрешить ввод нескольких значений или есть виды контактной информации с отключенным флажком Отображать всегда, то в окне для ввода предусмотрена кнопка +Телефон, адрес. При нажатии на нее предлагается список возможных видов контактной информации для добавления. Также если добавлены права на редактирование видов контактной информации, то с помощью кнопки Настроить можно перейти к настройкам видов контактной информации.
